Battery ladder — T2 / T3 / T4 power-storage progression

For agentic workers REQUIRED SUB-SKILL: use superpowers:subagent-driven-development. Steps use checkbox markers (the ☐ glyphs); flip to ✓ as you go.
Goal: add T2/T3/T4 battery buildings + generalise the existing single-battery (T5 Singularity Battery) energy buffer to a per-tier capacity table summed across all battery types on an island. New batteryCapacityMul skill effect lets the player scale capacity via Power Systems sub-path. Schema bump v11→v12 renames singularityStoredWs→batteryStoredWs.
Architecture: the existing charge/discharge integrator at economy.ts:1610-1693 already operates on one per-island energy buffer + one capacity value. Only the capacity calculation widens (per-def table summed) and the field is renamed. Battery buildings produce no W, consume no W; placement cost only (no Battery Factory crafted-unit chain).
Tech stack: Vite 5 + TypeScript strict + PixiJS 8 + vitest. Strict + noUncheckedIndexedAccess + noUnusedLocals.
Source of truth: design spec at docs/superpowers/specs/2026-05-24-battery-ladder-design.html (v1) — but the spec's capacities + standby values are SUPERSEDED by the locked numbers in this plan (see §Tier table).
Locked design choices (do NOT deviate):
  • Tier ramp: T2 + T3 + T4 (plus existing T5 unchanged in capacity).
  • All batteries have 0 W standby draw — including dropping singularity_battery from 100W → 0W for consistency.
  • Placement cost only — NO Battery Factory crafted-unit chain.
  • One new batteryCapacityMul SkillEffect kind + ONE notable using it (no keystone, no archetype filler chain).
  • State field rename: singularityStoredWs → batteryStoredWs (everywhere).

import type { BuildingDefId } from './building-defs.js';

/** Per-battery-def electrical buffer capacity in W-seconds. Summed across
 *  every operational battery on an island to compute total cap. Buildings
 *  not in this table contribute 0 — non-battery buildings ignore the path. */
export const BATTERY_CAPACITY_WS: Readonly<Partial<Record<BuildingDefId, number>>> = {
  battery_bank:        5_000      * 3600,
  capacitor_bank:      100_000    * 3600,
  flywheel_array:      2_000_000  * 3600,
  singularity_battery: 50_000_000 * 3600,
};

/** Total per-island battery capacity in W-seconds, summed across operational
 *  batteries × the skill-tree batteryCapacity multiplier. */
export function batteryCapacityWs(state: IslandState, mul: SkillMultipliers): number {
  let raw = 0;
  for (const b of state.buildings) {
    if (!isOperationalBuilding(b)) continue;
    const per = BATTERY_CAPACITY_WS[b.defId] ?? 0;
    raw += per;
  }
  return raw * mul.batteryCapacity;
}

 *  except the v literal AND the per-island state's energy-buffer field name. */

  extends Omit<SerializedIslandState, 'batteryStoredWs'> {

export interface SerializedSnapshotV11
  extends Omit<Omit<SaveSnapshot, 'v'>, 'islandStates'> {
  readonly v: 11;

    readonly id: string;

Step 3. Add the migration function:
/** Migrate a v11 snapshot to v12 (current). Renames the per-island energy
 *  buffer field singularityStoredWs → batteryStoredWs. Lossless — every
 *  island's stored energy carries over to the generalised battery system. */
export function migrateV11toV12(s: SerializedSnapshotV11): SaveSnapshot {
  return {
    ...s,
    v: 12 as const,
    islandStates: s.islandStates.map((entry) => {
      const { singularityStoredWs, ...stateRest } = entry.state;
      return {
        id: entry.id,
        state: {
          ...stateRest,
          batteryStoredWs: singularityStoredWs,
        } as unknown as SerializedIslandState,
      };
    }),
  } as SaveSnapshot;
}

describe('battery ladder integration', () => {
  it('Battery Bank charges from power surplus and caps at BATTERY_CAPACITY_WS', () => {
    const state = makeState({
      buildings: [
        { id: 'b1', defId: 'battery_bank', x: 0, y: 0 },
        { id: 'f1', defId: 'coal_furnace', x: 1, y: 0 },
        { id: 'm1', defId: 'mine', x: 2, y: 0 },

    // Advance ~5 minutes. Coal furnace produces W, mine consumes; if surplus > 0,
    // battery charges.
    advanceIsland(state, 300_000);
    expect(state.batteryStoredWs).toBeGreaterThan(0);
    // Cap is 5 kWh × 3600 = 18_000_000 Ws for one Battery Bank.

        { id: 'c1', defId: 'capacitor_bank', x: 1, y: 0 },
      ],

    const mul = blankMultipliers();

    const cap2 = batteryCapacityWs(state, mul);
