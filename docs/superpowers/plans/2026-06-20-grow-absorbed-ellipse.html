Goal & approach

Goal: let the player grow any constituent ellipse of a merged island (not just the primary) via the Land Reclamation Hub, choosing the lobe through a scrollable per-constituent picker, each lobe capped by its own origin biome.

Approach: a cap-only biome field on each extraEllipses entry (stamped at merge time, defaulted for legacy saves by a v27→v28 migration); the three land-reclamation.ts functions become constituent-indexed (index 0 = primary, N = extraEllipses[N-1]); a constituentIndex flows through the expand-island gateway method + server intent (server re-validates with the same pure functions); the inspector's Reclamation section becomes a scrollable row-per-constituent list, with a render-layer overlay drawing numbered badges on the map.

02

Key code — the engine generalization

The load-bearing change: cost and mutation keyed by constituent index. Index 0 reproduces today's primary behavior exactly; index N rebuilds extraEllipses[N-1] (entries are readonly).

src/land-reclamation.ts — gate + mutation (Task 4)
function constituentAxis(spec, index, axis) {       // radius + cap-biome of a constituent
  if (index === 0)
    return { current: axis === 'major' ? spec.majorRadius : spec.minorRadius, biome: spec.biome };
  const e = (spec.extraEllipses ?? [])[index - 1];
  if (!e) return null;                                 // → 'bad-constituent'
  return { current: axis === 'major' ? e.major : e.minor, biome: e.biome ?? spec.biome };
}

export function canExpandConstituent(spec, state, index, axis) {
  if (!hasLandReclamationHub(spec)) return { ok:false, reason:'no-hub' };
  const c = constituentAxis(spec, index, axis);
  if (!c) return { ok:false, reason:'bad-constituent' };
  const max = BIOME_MAX_RADII[c.biome][axis];        // per-lobe origin biome cap
  if (c.current >= max) return { ok:false, reason:'axis-at-max' };
  // …inventory check against landReclamationCost(spec, index, axis)…
  return { ok:true };
}

export function expandConstituent(spec, state, index, axis) {
  if (!canExpandConstituent(spec, state, index, axis).ok) return;
  // …deduct landReclamationCost(spec, index, axis)…
  if (index === 0) { axis === 'major' ? spec.majorRadius++ : spec.minorRadius++; }
  else {                                            // rebuild the readonly extra entry
    const e = spec.extraEllipses![index - 1]!;
    spec.extraEllipses![index - 1] = { ...e,
      major: axis === 'major' ? e.major + 1 : e.major,
      minor: axis === 'minor' ? e.minor + 1 : e.minor };
  }
}
