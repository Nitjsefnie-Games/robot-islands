Design spec · robot-islands · saltwater-cell battery bootstrap · 2026-05-26

Saltwater cell — a T2 home-buildable battery component

Context Battery Bank (T2, 5 kWh) is the smallest power buffer for night-time solar bridging — yet its placement cost { battery: 4, wire: 3, steel_beam: 1 } requires battery tokens that only come from the T3 Battery Factory, which itself needs lithium from the T3 Lithium Extractor, which has requiredTile: ['lithium_vein'] — biome-locked terrain only present on Volcanic islands. To build the cheapest battery a player must reach T3, drone-scout for a lithium_vein island, settle it via the Foundation Kit chain, and level it to T3 — at which point Capacitor Bank + Geothermal Vent + alternative T3 power sources make the Battery Bank pointless. This spec introduces a new T2 component (saltwater_cell) crafted at a new T1 building (Cell Press) using only home-Plains-available inputs, and rewrites Battery Bank's placement cost to use it. The T3 battery token stays as-is for upstream consumers (Capacitor Bank, Flywheel Array, Singularity Battery factory).

Decision
Add saltwater_cell (T2 component) + Cell Press (T1 building); rewrite battery_bank.placementCost from battery to saltwater_cell. Pure home-Plains buildable, zero discovery.
New resource: saltwater_cell — T2 fabricated component in the components storage category (same bucket as Wire, Bolt, Battery, Gear).
New building: cell_press — T1 manufacturing, single-tile 1×1 footprint, 20W power draw, placement cost { stone: 20, wood: 10 }. No requiredTile, no requiresHeat. Placeable anywhere on grass.
New recipe: at cell_press: 1 saltwater + 1 iron_ingot + 1 wire → 1 saltwater_cell, 40s cycle, category manufacturing.
Modified placement cost: battery_bank.placementCost changes from { battery: 4, wire: 3, steel_beam: 1 } to { saltwater_cell: 4, wire: 3, steel_beam: 1 }. Capacity unchanged at 5 kWh. The battery token is no longer referenced by any T2 building — it remains consumed by capacitor_bank (T3, 20), flywheel_array (T4, 100), and the singularity_battery factory recipes (T5).
Out of scope: any rework of battery production, lithium chain, Battery Factory placement. Higher-tier batteries (capacitor_bank, flywheel_array, singularity_battery) keep their existing T3+ recipes.
01

Why the inversion exists

Battery Bank ships at T2 (level 11+) but its battery-token requirement gates it behind a T3 chain. Full chain at HEAD as of 74cbc43:

ResourceProducerProducer tierGate
batteryBattery FactoryT3L25+ on the producer island
lithiumLithium ExtractorT3requiredTile: ['lithium_vein'] — biome-locked
rigid_plasticRigid Plastic PressT2chains through naphtha → Pump Jack on oil_well

Lithium_vein only generates in the Volcanic biome (biomes.ts:104 rareTerrain). Players must drone-scout, find a Volcanic island, settle it (Foundation Kit + Ship/Helicopter dispatch chain), level it to T3, then place the Lithium Extractor. By then they have Capacitor Bank (T3 buffer, 100 kWh) and Geothermal Vent (T1, day/night-independent on Volcanic). Battery Bank's 5 kWh storage is structurally obsolete by the time it's craftable.

04

New building — cell_press

A primitive cell-assembly building. Lives in the T1 manufacturing tier so it can be unlocked early; uses one T2 input (wire) which effectively gates output until the steel chain is online — but the building itself is placeable from L1.

cell_press: {
  id: 'cell_press',
  displayName: 'Cell Press',
  category: 'manufacturing',
  tier: 1,
  footprint: SHAPES.single,
  fill: 0x80a0c0,       // pale electrolyte blue
  stroke: 0x203040,
  power: { consumes: 20 },
  placementCost: { stone: 20, wood: 10 },
  glyph: '◫',
}

No requiredTile, no requiresHeat, no requiredBiomes, no gates, no adjacency buffs. Plain manufacturing slot.

BuildingDefId union at building-defs.ts gets 'cell_press' appended.

05

New recipe — saltwater cell assembly

cell_press: {
  cycleSec: 40,
  inputs: { saltwater: 1, iron_ingot: 1, wire: 1 },
  outputs: { saltwater_cell: 1 },
  category: 'manufacturing',
}

Recipe key matches the building defId (the codebase convention). Cycle time 40s is a balance choice: short enough to feel responsive when bootstrapping (4 cells in ~160s), long enough that the player still needs a couple of Cell Press buildings (or to wait) when ramping toward the 4 needed for a Battery Bank.

Throughput sanity-check: at 40s/cell and 1 Cell Press, 4 cells take 160s wall-clock. At 1 Coastal Pump (1 saltwater / 13s), 4 saltwater takes ~52s. At 1 Smelter (1 iron_ingot / 27s), 4 iron_ingot takes 108s. At 1 Metal Rolling Mill (1 wire / 133s), 4 wire takes 532s — the wire bottleneck dominates. Acceptable — wire is a known T2 chokepoint.

07

Tests

Resource catalog completeness — assert saltwater_cell appears in ALL_RESOURCES, RESOURCE_STORAGE_CATEGORY, XP_WEIGHT, and the inventory-ui category map. Existing resource-catalog tests already enforce these invariants by iterating the union; the new entry should slot in cleanly.
Cell Press recipe production — given an IslandState with all 3 inputs available, the existing tick-loop integration tests should produce saltwater_cell on advance. Add one new test mirroring an existing recipe-production test (e.g. the smelter or assembler test pattern).
Battery Bank placement — the existing placement-cost validation tests already iterate every building's placementCost; verify that the modified battery_bank placement cost passes inventory-shortfall checks under a stocked-inventory fixture with 4 saltwater_cell + 3 wire + 1 steel_beam.
Storage cap respect — saltwater_cell joining the components bucket means a Crate / Component Warehouse labeled components will accept it. Existing storage-cap tests should automatically cover this by virtue of the new resource being in the union; spot-check with a test that places a Crate, labels it components, and verifies saltwater_cell fills it.
No regressions on T3+ battery consumers — capacitor_bank, flywheel_array, and the singularity_battery factory recipe should continue to work exactly as before (they reference battery directly, not saltwater_cell).
08

Persistence

No schema bump required.

Inventory is serialized as Record<ResourceId, number>. Adding 'saltwater_cell' to the union doesn't change the serialized shape — older saves simply lack the key (which deserializes to 0 / undefined, treated as 0 by inv() helpers). New buildings get a defId added to the type union; existing saves don't reference it. Recipe table is in-memory only.

A v14 save loaded under the new code starts seeing saltwater_cell available with 0 stock; the player can place a Cell Press immediately and begin production. Buildings table is open-set — existing serialized buildings (none of which are cell_press) just keep working.

09

Risks

Risk
Sev
Mitigation
Adding a new ResourceId without updating every catalog table (RESOURCE_STORAGE_CATEGORY, XP_WEIGHT, inventory-ui map) causes runtime undefined-access or missing-storage-category errors.
MED
Existing tests iterate the ResourceId union and assert presence in each map. Adding saltwater_cell to one without the others fails tests at the catalog-completeness check. Update all four touchpoints in one commit.
Procgen Plains colonies (settled, not home) have no water tile, so Cell Press can't make saltwater_cell there without imported saltwater via routes.
LOW
Expected. The home island bootstrap is the primary use case for Battery Bank; downstream colonies typically settle on Forest or Coast biomes (both with water tiles in rareTerrain) by player choice. If a Plains colony lacks water, the player either imports saltwater via a route or settles a water-bearing neighbor.
Players who already built capacitor_bank / flywheel_array can no longer demolish-and-rebuild without the battery input chain. Battery Bank for those players becomes a strict downgrade.
LOW
By the time a player has capacitor_bank online, Battery Bank's 5 kWh is irrelevant compared to 100 kWh. The change makes Battery Bank useful for new/early players without affecting end-game.
Flavor incongruity: "saltwater_cell" vs the existing realistic chemistry tree (chlor_alkali, sulfuric_acid_plant, naphtha_cracker — all named for real industrial processes). A saltwater galvanic cell is technically real but novelty.
LOW
Real-world precedent: saltwater batteries (e.g. Aquion Energy) exist as commercial products. Naming consistent with the existing "Coal Furnace" / "Coke Oven" idiom — descriptive, not branded.
10

Out of scope

Reworking the battery token / Battery Factory tier
battery stays T3 lithium-ion. Battery Factory stays T3. Capacitor Bank + Flywheel Array + Singularity Battery factory continue consuming battery as before. This spec is additive — it gives the early game a path to Battery Bank without disturbing the T3+ chain.
Lithium chain de-gating
Lithium Extractor stays T3 + lithium_vein-locked. If a future spec wants to relax that gate (e.g. T2 brine extraction), it stands on its own.
Battery Bank capacity rebalance
Stays at 5 kWh. Capacitor Bank (100 kWh) remains the next upgrade tier.
Cell Press as a recipe-bearing notable
The building has one recipe (this one). No alternate recipes, rotations, or adjacency mechanics — keep it simple. Future variants (e.g. a "fluid_cell" using fresh_water + lubricant) could ship as separate recipes if needed.
